# 🚀 Space Invasion 2942

Un videojuego 2D tipo *space shooter* desarrollado en Processing 4, donde el jugador controla una nave espacial que debe eliminar enemigos, esquivar proyectiles y recoger power-ups para sobrevivir el mayor tiempo posible.

---

## ✨ Características

- Combate en tiempo real con registro de puntaje
- Tres tipos de power-ups: **vida**, **escudo** y **ametralladora**
- Sistema de dificultad progresiva
- Nave aliada con movimiento autónomo
- Efectos visuales de explosión y fondo espacial animado
- Historial de puntuaciones guardado en `.csv`
- Gráficos de estadísticas al finalizar la partida

---

## 🕹️ Controles

| Acción | Control |
|---|---|
| Mover nave | Mouse |
| Disparar | Click izquierdo |
| Usar Power-Up | Click derecho |
| Pausar | `esc` |

---

## 📋 Requisitos

- [Processing 4]
- Librería Sound — cómo instalarla: Sketch > Import Library > Add Library > buscás "sound"
---

## ▶️ Cómo ejecutar

1. Cloná el repositorio:
   ```bash
   git clone https://github.com/BohlSan/juego-2942-processing.git
   ```
2. Abrí Processing 4
3. Desde Processing, abrí el archivo principal: `SpaceInvasion/SpaceInvasion.pde`
4. Presioná el botón **Run** (▶)
